# Pastebin Lite

A minimal Pastebin-like application built with Next.js and Redis.

## Features
- Create and share text pastes
- Optional expiration time (TTL)
- Optional max view count
- Redis-backed persistence
- Deployed on Vercel

## Tech Stack
- Next.js (App Router)
- Upstash Redis
- Vercel

## Running Locally
```bash
npm install
npm run dev
